## ğŸ¨ **Customization Guide**

### Personal Information
1. **Update Profile Details** in `index.html`:
   - Name, title, and contact information
   - Profile image paths
   - Social media links

2. **Modify Experience & Education**:
   - Update timeline entries in the Resume section
   - Add or remove work experience items
   - Update education details

3. **Customize Skills**:
   - Modify skill names and percentages
   - Add or remove skills as needed
   - Update progress bar values

### Projects
1. **Add New Projects**:
   - Add project images to `assets/images/`
   - Update project list in HTML
   - Modify project descriptions and links

2. **Update Project Images**:
   - Replace existing images with your project screenshots
   - Maintain aspect ratios for best display
   - Use high-quality images (recommended: 800x600px)

### Styling
1. **Color Scheme** - Modify CSS variables in `style.css`:
   ```css
   :root {
     --orange-yellow-crayola: #your-color;
     --vegas-gold: #your-color;
     /* ... other color variables */
   }
   ```

2. **Typography** - Update font families and sizes:
   ```css
   --ff-poppins: 'Your-Font', sans-serif;
   ```

3. **Layout** - Adjust spacing and grid layouts as needed
